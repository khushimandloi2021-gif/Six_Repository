# Rock Paper Scissor Game 🎮

A fun terminal-based Rock Paper Scissor game with custom ASCII art console design!

## How it works
- User enters their choice: rock, paper, or scissor
- Computer randomly picks its choice using random module
- Program compares both choices and declares the winner

## Rules
- Rock beats Scissor
- Paper beats Rock
- Scissor beats Paper
- Same choice = Tie

## Concepts used
- Random module
- if-elif-else conditions
- String comparison
- Nested conditionals
